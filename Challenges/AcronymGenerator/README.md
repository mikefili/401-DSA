# Acronym Generator

## Challenge
Write a function that will take in a string and return the acronym of that string. Words that cannot go in the acronym: ["a", "for", "an", "and", "of", "or", "the", "to", "with"]

## Approach & Efficiency
- Time: O(n)
- Space: O(1)
